# ease-text-truncate

A utility class that truncates overflowing text with an ellipsis on a single line.

## Usage
```html
<p class="ease-text-truncate" style="max-width: 300px;">
  This is a very long text that will be truncated.
</p>
```

## Properties
- `overflow: hidden`
- `white-space: nowrap`
- `text-overflow: ellipsis`
